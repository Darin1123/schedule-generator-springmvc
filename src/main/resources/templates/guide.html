How to add/modify a course 1. Write a Course Name The course names should follow a pattern. Start with faculty name and follow with course code. For example, "ENG 1EE0", "COMPSCI 4ML3" are eligible course names.
2. Write a Term Name A term name should start with a year and end with a term (one of "spring", "summer", "fall", "winter"). For example, "2020 winter", "2019 spring" are eligible term names.
3. Write Course Sections Let's learn from an example. The key information that should be included for a section is weekyday(s), start time, and end time. In the example, the days are "Tu" and "Th", the start time is "09:00", and the end time is "12:00". THEREFORE, for this section, the correct input should be "TuTh 09:00-12:00" In general, each section should be one line. If the section are given in different days with different time, then seperate them with a comma. For another example, if there is a section have classes on Mondays and Tuesdays starting at 9:30 and ending at 10:20, and another class on Friday starting at 12:30-13:20. Then the input should be "MoTu 09:20-10:20, Fr 12:30-13:20".
